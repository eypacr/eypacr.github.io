Hakkımda

2024 yılında İnönü Üniversitesi'nde Bilgisayar Mühendisliği lisans eğitimimi tamamladım. C#, ASP.NET Core MVC, Web API ve SignalR gibi teknolojilerle modern yazılım çözümleri geliştirme konusunda deneyimliyim.

PostgreSQL ve Microsoft SQL Server (MSSQL) gibi veritabanı yönetim sistemlerinde verimli çözümler sunarak, yüksek performanslı uygulamalar oluşturdum. HTML, CSS ve Bootstrap kullanarak kullanıcı dostu web arayüzleri tasarladım. RESTful API ve JSON Web Token (JWT) ile güvenli sistemler geliştirdim. Teknolojiye olan tutkum ve problem çözme becerilerimle, yazılım dünyasına değer katmayı hedefliyorum.

Yaptığım İşler

  • Web Tasarımı

    En modern ve kullanıcı dostu tasarımlar, markanızın dijital kimliğini güçlendirecek şekilde profesyonel seviyede yapılır. Her bir projede estetik ile işlevselliği mükemmel bir şekilde harmanlıyorum.

  • Web Geliştirme

    Yüksek kaliteli, hızlı ve güvenli web siteleri geliştirme konusunda deneyimliyim. İhtiyaçlarınıza uygun web çözümleri sunarak, sitenizin performansını en üst düzeye çıkarıyorum.

  • Mobil Uygulamalar

    iOS ve Android platformlarında, kullanıcı dostu ve işlevsel mobil uygulamalar geliştiriyorum. İş süreçlerinizi hızlandıracak ve kullanıcı deneyimini iyileştirecek çözümler sunuyorum.

  • Fotoğrafçılık

    Her türlü etkinlik, portre ve ürün fotoğrafçılığı için yüksek kaliteli çekimler yapıyorum. Profesyonel fotoğraflarla markanızın görsel kimliğini güçlendirebilirsiniz.
